🔬 Trust Wallet — Systematic QR Test Suite

⚠ DISPLAY vs EXECUTION TEST

These QR codes are designed to test whether Trust Wallet shows one amount but executes a different amount. For testing purposes:
DISPLAY target: 1 USDT  |  EXECUTE target: 5 USDT
If any test executes 5 instead of 1, the display/execution mismatch is confirmed.

📋 How to test

  1. Scan INSIDE Trust Wallet — use the main QR scanner, not the Send → Scan flow
  2. For each test, note what the send screen shows for amount and recipient
  3. If it shows the correct display target (1 USDT), check what actually executes
  4. Priority order: trust:// tests first, then Deep Link variants
  5. Mark each as: ✅ Shown correct / ❌ No transfer / 🔄 Uses wrong amount
🆕 = Never tested  |  ❌ = Already failed in prior test  |  ⚠ = Mismatch expected
